Crypto Checker [ Incredible stupid & slow ! ]

Usage: cc.com [switches] ExeFile [>LogFile]

-i      - PE Info. For PE files only
-d      - Dump Mode
-x      - Dump with Extra Scan
-#nnn   - Scan for sect. nnn (in Dec). For PE files only
-o      - Scan Overlay (if any)
-a      - Scan ALL parts
-$name  - Check for specified algorithm name only
